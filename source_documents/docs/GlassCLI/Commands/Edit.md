# **Glass Edit Command**

### **`glass edit`**
#### **Description**
Convenience command to open the GlassCLI Project for editing
#### **Options**
- `--glass-repo-root`: The path to the root of the glass repo, defaults to the current directory. 
#### **Flags**
- `-v`, `--verbose`: Enables verbose logging for the command.
#### **Example Usage**
```
glass edit
```
